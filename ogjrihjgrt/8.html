    HTML5 предоставляет ряд глобальных атрибутов, которые могут быть использованы практически со всеми элементами.
    Эти атрибуты помогают добавлять метаданные, контекст и функциональность к элементам. Некоторые из основных глобальных атрибутов включают:

    class

    Этот атрибут используется для назначения класса стилям элемента. Классы применяются в CSS для определения стилей группы элементов.

    id

    Атрибут id уникально идентифицирует элемент на странице. Его значение должно быть уникальным в пределах всего документа.

    style

    Атрибут style позволяет задавать стилевые свойства непосредственно в HTML, избегая необходимости использования внешних CSS файлов.

    lang

    Атрибут lang указывает язык элемента. Часто используется совместно с dir для задания направления текста (например, lang="en" для английского языка и dir="ltr" для слева направо).

    title

    Атрибут title содержит дополнительную информацию о элементе, которая часто появляется как всплывающая подсказка при наведении курсора мыши.
